# 업데이트 내용

## GitHub Actions 상세 보기

- 커밋 옆 Actions 아이콘을 누르면 같은 커밋에서 실행된 모든 워크플로와 Job/Check 상태를 한 팝업에서 확인할 수 있습니다.
- 각 워크플로와 Job/Check를 선택하면 해당 GitHub 실행 페이지로 바로 이동합니다.

## 사용성 개선

- Actions 상태를 아이콘으로 간결하게 표시하고 커밋 열 오른쪽에 고정해, 브랜치나 태그가 많아도 다른 열이 밀리지 않습니다.
- 클릭 가능한 Actions 항목 위에서 포인터 모양이 바뀌어 이동 가능한 요소를 쉽게 구분할 수 있습니다.
