# 업데이트 내용

## 브랜치 동기화

- 로컬 브랜치에서 upstream과의 ahead/behind 차이를 서로 다른 색상으로 확인할 수 있습니다.
- 현재 체크아웃된 브랜치를 rebase 방식으로 pull하고 upstream으로 push할 수 있습니다.
- `⌘R`을 누르면 열린 모든 저장소의 원격 변경 사항을 가져오고 로그를 새로고침합니다.

## 커밋 그래프

- 현재 `HEAD` 위치를 그래프와 배지로 명확하게 표시합니다.
- 커밋되지 않은 변경 사항을 그래프 상단의 `작업 중` 항목으로 표시합니다.
- 커밋 우클릭으로 브랜치와 태그를 확인하고 해시 또는 메시지를 복사할 수 있습니다.
- 현재 보이는 그래프 레인에 맞춰 그래프 영역의 폭을 조절합니다.
- 그래프 최상단에 작은 여백을 추가해 첫 항목과 헤더를 자연스럽게 구분합니다.

## 사용성 및 배포

- Git 로그 초기 로딩과 새로고침 상태를 명확하게 표시하고 화면 전환을 부드럽게 개선했습니다.
- 서명과 Apple 공증을 마친 DMG 설치 파일을 GitHub Release에서 제공합니다.
