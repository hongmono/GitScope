macOS용 Git 이력 뷰어

브랜치를 읽는 데 드는 시간을 줄이세요.

여러 저장소의 커밋, 브랜치, 작업 상태를 한 화면에서 읽고 안전하게 동기화하세요.

최신 DMG 받기 ↘ 소스 보기
Apple Silicon macOS 14+ 공증 완료 Sparkle 업데이트

01 / FOCUS

흩어진 Git 상태를
하나의 시야로.

탭을 오가며 명령어를 반복하기 전에, 여러 저장소의 흐름과 현재 위치를 한 번에 파악할 수 있습니다.

02 / WORKFLOW

평소 쓰던 Git 흐름 그대로.

읽고, 판단하고, 필요한 순간에만 동기화합니다.

01

OPEN

저장소는 여러 개, 맥락은 하나

저장소별 탭을 열고 ⌘1부터 ⌘9까지 즉시 이동하세요. 종료한 뒤에도 탭이 복원되고, 비활성 로그는 메모리에서 내려갑니다.

02

READ

HEAD와 작업 상태를 그래프 안에서

현재 위치, 브랜치, 태그, 커밋되지 않은 파일을 같은 타임라인에서 봅니다. upstream과의 ahead, behind 차이도 색으로 구분됩니다.

03

SYNC

rebase pull과 push를 바로

브랜치를 우클릭해 현재 브랜치를 rebase pull하거나 upstream으로 push하세요. ⌘R은 모든 저장소에서 fetch를 실행합니다.

03 / SIGNAL

필요한 신호만 남겼습니다.

작업 중
커밋되지 않은 파일을 그래프 최상단의 현재 작업으로 표시합니다.
현재 위치
HEAD와 연결된 브랜치, 태그를 커밋 옆에서 바로 확인합니다.
원격 차이
upstream보다 앞선 커밋과 뒤처진 커밋을 서로 다른 색으로 읽습니다.

04 / GUARDRAILS

Git을 바꾸기 전에,
Git을 읽는 도구.

GitScope는 이력을 이해하고 현재 브랜치를 안전하게 동기화하는 데 집중합니다.

포함 fetch · pull --rebase · push
미포함 commit · checkout · merge · force push
