rebase pull과 push

Signal over noise

그래프 안에서
지금 상태까지.

현재 위치와 작업 중인 파일, Actions 결과를 타임라인 안에 표시해 다음 행동에 필요한 맥락을 놓치지 않습니다.

현재 위치
HEAD와 연결된 브랜치, 태그
앞선 변경
push할 커밋을 초록색으로
뒤처진 변경
가져올 커밋을 주황색으로
Actions 상태
